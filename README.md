# VoiceInk Installer — бесплатная диктовка для Mac

Скрипт ставит **open-source VoiceInk** на твой Mac. Это бесплатная локальная альтернатива Superwhisper ($84/lifetime) и Wispr Flow ($15/мес).

## Почему этому источнику можно доверять

VoiceInk — проект разработчика **Prakash Joshipax**, открыт на GitHub под GPL v3:

| Метрика | Значение |
|---|---|
| ⭐ Звёзды на GitHub | **4,788** |
| 🍴 Форки | **652** |
| 📅 Возраст проекта | с октября 2024 (~1.5 года) |
| 🛠 Активность | обновления каждую неделю |
| 📜 Лицензия | GPL v3 (open-source, copyleft) |

Это **не однодневный скрипт от незнакомца**. Это популярный Mac-проект с активной аудиторией, исходный код можно прочитать и проверить целиком. Скрипт-инсталлер ниже **только автоматизирует сборку**, не меняя ни строчки кода VoiceInk.

🔗 Оригинал: https://github.com/Beingpax/VoiceInk

## Что получишь

Зажал хоткей → сказал в микрофон → отпустил → текст вставился в активное поле (Telegram, Notes, любое окно).

- ✅ Бесплатно навсегда (open-source GPL v3)
- ✅ Всё локально, ничего не уходит в облако
- ✅ Поддержка русского
- ✅ Работает на M1 / M2 / M3 / M4 без подписки

## Что нужно

- macOS **14.4** или новее
- **30+ ГБ** свободного места на диске
- **Apple ID** (для скачивания Xcode)
- **Xcode** из App Store (~10 ГБ, бесплатно) — поставить **до** запуска скрипта
- Терпение на ~30 минут

## Установка

### Шаг 1. Поставь Xcode

Открой **App Store** → поиск **Xcode** → **Get** → **Install**. Загрузка ~10 ГБ, около 15–30 минут.

### Шаг 2. Скачай и запусти скрипт

В Терминале:

```bash
curl -fsSL https://raw.githubusercontent.com/disciplineiswin/voiceink-installer/main/install.sh -o /tmp/voiceink-install.sh
chmod +x /tmp/voiceink-install.sh
/tmp/voiceink-install.sh
```

Скрипт сам:
1. Поставит Homebrew и cmake (если нет)
2. Переключит инструменты на полный Xcode
3. Склонирует VoiceInk и соберёт `.app`
4. Предложит выбрать модель (по умолчанию — Parakeet v3, качается прямо в VoiceInk)
5. Поставит всё в `/Applications/`
6. Запустит VoiceInk

Спросит пароль Mac пару раз — это нормально.

### Шаг 3. Настрой VoiceInk

После запуска приложения:

1. **AI Models** → выбери модель → дождись загрузки → **Set as Default**
   - **Parakeet v3** — бери его: быстрый, лёгкий, русский из коробки (VoiceInk сам докачает)
   - **Large v3 Turbo (Quantized)** — запасной Whisper-вариант, тоже хорош для русского
2. **Transcription Language** → **Russian**
3. **App Permissions** → дай три разрешения:
   - Microphone
   - Accessibility — вставка текста в поле
   - Input Monitoring — чтобы хоткей работал в любом приложении
4. **Settings → Hotkey** → выбери клавишу. Надёжнее всего **правый ⌥ Option** или **F5** — fn часто занят сменой языка (см. раздел ниже, если хоткей молчит)

### Шаг 4. Проверь

Открой Telegram → курсор в поле сообщения → зажми хоткей → скажи «тестовое сообщение» → отпусти. Текст появится через секунду.

## Что ещё умеет (по желанию)

Базовой диктовки хватает большинству, но если захочется большего:

### 🎧 Распознать готовый аудио- или видеофайл

Боковое меню → **Transcribe Audio** → перетащи файл в окно → **Start**. Через минуту получишь текст. Удобно для голосовых, подкастов, записей встреч, интервью.

Форматы: WAV, MP3, M4A, AIFF, MP4, MOV, AAC, FLAC, OGG, OPUS и другие.

### 🤖 Авто-исправление текста (Enhancement)

VoiceInk умеет прогонять расшифровку через языковую модель — она **убирает «эээ», расставляет запятые, чинит грамматику и форматирует**. Включается в **Enhancement → Enable Enhancement**, а кто именно правит текст — выбираешь в **AI Provider Integration → Provider**. Два удобных пути:

**Проще всего — Groq (облако, бесплатно):**
1. **Enhancement → Provider → Groq**
2. **Get API Key** → откроется сайт Groq, заведи бесплатный ключ → вставь в **API Key** → **Verify and Save**
3. Выбери **Model** из списка, а ниже в **Enhancement Prompts** — режим (**Default** / **Assistant**)

> В облако уходит **только текст расшифровки** (не голос). Для обычных заметок нормально; если текст чувствительный — бери вариант ниже.

**Полностью локально — Ollama (ничего не уходит с Mac):**
1. Поставь Ollama и скачай модель (в Терминале):
   ```bash
   brew install ollama
   ollama serve &              # запустить в фоне
   ollama pull qwen2.5         # хорошо знает русский (или llama3.1 / gemma3)
   ```
2. **Enhancement → Provider → Ollama** — адрес уже прописан (`http://localhost:11434`), API-ключ не нужен → в **Model** выбери скачанную модель.

> Минус локального варианта: качает пару ГБ под модель и сильнее грузит процессор. Голой диктовки Parakeet v3 большинству и так достаточно.

Включать/выключать исправление на лету можно отдельным хоткеем в **Settings → Hotkey**.

### Ещё по мелочи

- **Power Mode** — сам подбирает настройки и промпт под активное приложение (в почте — деловой тон, в чате — разговорный).
- **Dictionary** — словарь: добавь имена, термины и бренды, чтобы их не коверкало, плюс правила авто-замены.
- **History** — вся история расшифровок под рукой, можно скопировать заново.

> 📖 Полная документация по всем возможностям: https://tryvoiceink.com/docs

## Хоткей не срабатывает

Зажимаешь клавишу, говоришь — а текст не появляется. Три причины по частоте:

### 1. Клавиша fn занята переключением языка

Если у тебя **несколько раскладок** (русская + английская и т.д.), macOS по умолчанию вешает на клавишу **🌐 fn** действие «сменить источник ввода». Тогда нажатие fn перехватывает система, и до VoiceInk оно не доходит.

Освободи fn:
1. **System Settings → Keyboard** → строка **«Press 🌐 key to»** → поставь **Do Nothing**
2. Язык переключай иначе: там же включи **Ctrl + Space** или **Caps Lock** для смены раскладки

Признак именно этой проблемы: при нажатии fn у тебя меняется язык ввода вместо записи.

> Совет: fn — самая «системная» клавиша на маке, macOS любит отбирать её после обновлений. Если не хочешь за этим следить — назначь в VoiceInk **правый ⌥ Option** или **F5**: их система не трогает.

### 2. Слетело разрешение Input Monitoring

VoiceInk ловит хоткей глобально через **Input Monitoring** и **Accessibility**. При каждой пересборке `.app` подпись меняется, и macOS сбрасывает эти разрешения — хоткей замолкает, хотя микрофон и системная диктовка Apple работают.

Сбрось все разрешения VoiceInk начисто и выдай заново:

```bash
tccutil reset Accessibility com.prakashjoshipax.VoiceInk
tccutil reset ListenEvent com.prakashjoshipax.VoiceInk    # это и есть Input Monitoring
tccutil reset Microphone com.prakashjoshipax.VoiceInk
tccutil reset PostEvent com.prakashjoshipax.VoiceInk      # вставка текста
```

Потом перезапусти VoiceInk и включи всё в **System Settings → Privacy & Security**:
- **Input Monitoring** → VoiceInk → вкл ← главное для хоткея
- **Accessibility** → VoiceInk → вкл
- **Microphone** → VoiceInk → вкл

### 3. После переустановки разрешение «залипло» на старой версии

Если ты **пересобрал/обновил** VoiceInk, а в списке Privacy он уже стоит включённым, но всё равно не работает — macOS держит запись от старой подписи. Надо удалить и добавить заново:

1. **System Settings → Privacy & Security → Input Monitoring** (и так же Accessibility)
2. Выдели **VoiceInk** в списке → нажми **`–`** (убрать)
3. Нажми **`+`** → добавь заново `/Applications/VoiceInk.app`
4. Перезапусти VoiceInk

Только после удаления старой записи и добавления свежей хоткей оживёт.

## Обновить VoiceInk

Скрипт сам подтягивает свежую версию. Запусти его повторно:

```bash
/tmp/voiceink-install.sh
```

Заменит установленный VoiceInk на актуальный билд.

## Бенчмарки моделей и под какое железо что брать

| Модель | Размер | Качество русского | Скорость | RAM | Под какое железо |
|---|---|---|---|---|---|
| **Parakeet v3 (FluidAudio)** | ≈494 МБ (в app) | **★★★★★** | **очень быстро (ANE)** | **низкая** | **Apple Silicon M1+ — default** |
| ggml-small | 465 МБ | ★★★ | ×3 реалтайма | ~1 ГБ | старые Intel-маки |
| Large v3 Turbo Quantized | 547 МБ | ★★★★★ | ×1.5 реалтайма | ~2 ГБ | Whisper-вариант для M1/M2 8 ГБ |
| Large v3 Turbo (full) | 1.5 ГБ | ★★★★★ | ×0.8 реалтайма | ~3 ГБ | M1/M2 16+ ГБ, M3, M4 |
| Large v3 (full) | 2.9 ГБ | ★★★★★ | ×0.16 реалтайма | ~4 ГБ | M3 Pro / M4 Pro+ |

**Скрипт по умолчанию предлагает Parakeet v3** на Apple Silicon — он быстрее Whisper, легче по RAM и докачивается прямо в VoiceInk (curl не нужен). При запуске спросит, не хочешь ли ты другую модель. На Intel-маках дефолт — Whisper Large v3 Turbo Quantized (547 МБ).

**Если у тебя M3 / M4 / 16+ ГБ RAM** — стоит взять **Large v3 (full)** или **Large v3 Turbo (full)**: точность выше, скорость на твоём железе всё равно комфортная.
